## 🌐 Overview

**What Websites Know About You** is a privacy awareness project developed by **PKER05 Labs** to demonstrate the information websites can collect when a user simply visits a webpage.

The project shows the difference between:

* ✅ Information directly available to websites
* 🧠 Information that can be inferred from collected signals
* 🔒 Information that requires explicit user permission

The goal is to help users understand their digital footprint and make better privacy decisions online.

---

# 🎯 Vision

The internet constantly exchanges information between users and services. Many people do not realize how much information their browser and network reveal automatically.

This project aims to make privacy concepts understandable through a practical, interactive experience.

Instead of explaining tracking theoretically, users can see real examples of:

* Network information
* Browser details
* Device capabilities
* Browser fingerprinting concepts
* Privacy risks

---

# ✨ Features

## 🌐 Network Information

Displays information available through network analysis:

* Public IP address
* Approximate location from IP data
* Internet Service Provider
* Network organization
* Autonomous System Number (ASN)

> Network information depends on external IP intelligence services and may vary depending on network configuration.

---

## 💻 Browser Information

Collects browser-provided information:

* User-Agent
* Browser platform
* Preferred language
* Timezone
* Cookie availability
* Do Not Track status
* Online/offline status

---

## 🖥 Device Information

Displays available device details:

* Screen resolution
* Available screen size
* Pixel ratio
* CPU core estimate
* Memory estimate
* Touch support

---

## 🧩 Browser Capabilities

Tests supported browser technologies:

* Local Storage
* Session Storage
* IndexedDB
* WebGL
* WebAssembly
* Browser feature availability

---

## 🧠 Privacy Inference System

Explains what websites may estimate using available signals.

Examples:

* Possible device type
* Browser uniqueness
* Technical profile estimation
* Privacy exposure level

> These are estimates and are not guaranteed facts.
